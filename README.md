
# データ構造とアルゴリズム

### アルゴリズムの効率評価
- 時間計算量(プログラムの実行に必要な時間を評価)
- 領域計算量(プログラムの実行に必要な領域を評価)

### 計算量O(オーダ)
- O(n)/O(logN)/O(NlogN)はn増加でも計算量は増減しないので効率◯
- O(n^2)はn^2に比例するので効率×
